## Overview

The Farming system allows players to plant seeds in fixed world patches, watch crops grow through multiple stages in real-time, and harvest produce for Farming XP. Each player has their own instanced state per patch, so multiple players can use the same patch simultaneously without interference.

<Note>
Farming patches are RuneScape-style allotment patches. The system supports multiple patch locations across the game world, with each location belonging to a specific plot that can be unlocked.
</Note>

## Core Components

The farming system is implemented across three main modules:

- **farming.rs** - Main system structure and configuration loading
- **farming/patches.rs** - Patch management, planting, harvesting, and growth mechanics
- **farming/contracts.rs** - Farming contracts with difficulty tiers and rewards

## Growth Stages

Crops grow through multiple stages based on elapsed time. The growth stage is calculated dynamically:

```rust
pub fn growth_stage(&self, crops: &HashMap<String, CropConfig>, current_time: u64) -> u32 {
    if let Some(crop) = crops.get(&self.crop_id) {
        let elapsed_ms = current_time.saturating_sub(self.planted_at);
        let total_ms = (crop.growth_time_minutes * 60.0 * 1000.0) as u64;
        
        if total_ms == 0 {
            return crop.growth_stages;
        }
        
        // Calculate current stage as percentage of total growth time
        let stage = (elapsed_ms as f64 / total_ms as f64 * crop.growth_stages as f64) as u32;
        stage.min(crop.growth_stages)
    } else {
        0
    }
}
```

### Growth Tick System

The system periodically checks for growth stage changes and broadcasts updates to players:

```rust
pub fn tick_growth(&mut self, current_time: u64) -> Vec<(String, PatchUpdate)> {
    let mut updates = Vec::new();
    
    for ((patch_id, player_id), state) in self.player_states.iter_mut() {
        let current_stage = state.growth_stage(&self.crops, current_time);
        
        // Only broadcast when stage changes
        if current_stage != state.last_broadcast_stage {
            state.last_broadcast_stage = current_stage;
            
            let is_harvestable = state.is_harvestable(&self.crops, current_time);
            let state_str = if is_harvestable {
                "harvestable"
            } else {
                "growing"
            };
            
            updates.push((
                player_id.clone(),
                PatchUpdate {
                    patch_id: patch_id.clone(),
                    state: state_str.to_string(),
                    crop_id: state.crop_id.clone(),
                    growth_stage: current_stage,
                    owner_id: player_id.clone(),
                },
            ));
        }
    }
    
    updates
}
```

## Harvesting Crops

When a crop reaches full maturity, players can harvest it for produce and XP:

<Steps>
  <Step title="Validate patch state">
    Ensure the patch exists and has a crop ready to harvest.
    
    ```rust
    let key = (patch_id.to_string(), player_id.to_string());
    let state = self.player_states.get(&key)
        .ok_or_else(|| "Nothing to harvest".to_string())?;
    
    if !state.is_harvestable(&self.crops, current_time) {
        return Err("This crop is not ready to harvest yet".to_string());
    }
    ```
  </Step>
  
  <Step title="Calculate harvest amount">
    Random yield within configured range, with bonus for over-leveled players.
    
    ```rust
    let mut rng = rand::thread_rng();
    let base_amount = rng.gen_range(crop.harvest_amount_min..=crop.harvest_amount_max);
    
    // Bonus: +1 per 10 levels above requirement
    let bonus = ((farming_level - crop.level_required) / 10).max(0);
    let amount = base_amount + bonus;
    ```
  </Step>
  
  <Step title="Calculate XP and seed return">
    XP scales with harvest amount. Seed return is a random chance.
    
    ```rust
    let xp = crop.xp_per_harvest * amount as i64;
    
    let seed_returned: f32 = rng.gen_range(0.0..1.0);
    let seed_returned = seed_returned < crop.seed_return_chance;
    ```
  </Step>
  
  <Step title="Clear patch and return rewards">
    Remove the patch state and return harvest results.

    ```rust
        xp_gained: xp,
    ```

```rust
// Check if a player has unlocked a specific plot
pub fn is_plot_unlocked(&self, player_id: &str, plot_id: u32) -> bool {
    if plot_id <= 1 {
        return true;  // Plot 1 is always unlocked
    }
    self.player_plot_unlocks
        .get(player_id)
        .map(|plots| plots.contains(&plot_id))
        .unwrap_or(false)
}

// Unlock a plot for a player
pub fn unlock_plot(&mut self, player_id: &str, plot_id: u32) {
    self.player_plot_unlocks
        .entry(player_id.to_string())
        .or_default()
        .insert(plot_id);
}
```

### Recording Harvest Progress

When a player harvests crops, the system automatically checks for matching contracts:

```rust
pub fn record_contract_harvest(
    &mut self,
    player_id: &str,
    crop_id: &str,
    amount: i32,
) -> Option<(i32, i32, bool)> {
    let contract = self.contracts.get_mut(player_id)?;
    
    // Ignore if wrong crop or already complete
    if contract.crop_id != crop_id || contract.is_complete() {
        return None;
    }
    
    // Update progress, capped at requirement
    contract.amount_harvested = 
        (contract.amount_harvested + amount).min(contract.amount_required);
    
    let complete = contract.is_complete();
    
    // Returns (current, required, is_complete)
    Some((
        contract.amount_harvested,
        contract.amount_required,
        complete,
    ))
}
```

```rust
pub fn get_player_patch_states(&self, player_id: &str, current_time: u64) 
    -> Vec<PatchUpdate> {
    let mut updates = Vec::new();
    
    for patch in self.patches.values() {
        // Only include patches from unlocked plots
        if !self.is_plot_unlocked(player_id, patch.plot) {
            continue;
        }
        
        let key = (patch.id.clone(), player_id.to_string());
        if let Some(state) = self.player_states.get(&key) {
            // Player has a crop planted here
            let stage = state.growth_stage(&self.crops, current_time);
            let is_harvestable = state.is_harvestable(&self.crops, current_time);
            
            updates.push(PatchUpdate {
                patch_id: patch.id.clone(),
                state: if is_harvestable { "harvestable" } else { "growing" },
                crop_id: state.crop_id.clone(),
                growth_stage: stage,
                owner_id: player_id.to_string(),
            });
        } else {
            // Empty patch
            updates.push(PatchUpdate {
                patch_id: patch.id.clone(),
                state: "empty".to_string(),
                crop_id: String::new(),
                growth_stage: 0,
                owner_id: String::new(),
            });
        }
    }
    
    updates
}
```
