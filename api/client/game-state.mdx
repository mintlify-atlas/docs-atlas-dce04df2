## Overview

The `GameState` struct is the central data structure that holds all client-side game state, including player data, entities, UI state, network status, and world chunks. It coordinates between the network layer, input handling, and rendering systems.

## GameState

Main state container for the game client.

<ParamField path="local_player_id" type="Option<String>">

</ParamField>

## Example Usage

```rust
// Access local player
if let Some(player) = state.get_local_player() {
    println!("Player at ({}, {})", player.x, player.y);
    println!("HP: {}/{}", player.hp, player.max_hp);
    println!("Combat Level: {}", player.combat_level());
}

// Update camera to follow player
if let Some(player) = state.get_local_player() {
    state.camera.x = player.x;
    state.camera.y = player.y;
}

// Check connection status
if state.connection_status == ConnectionStatus::Connected {
    // Send commands to server
}

// Access chat messages
for message in state.ui_state.chat_messages.channel(&ChatChannel::Local) {
    println!("[{}] {}: {}", message.timestamp, message.sender_name, message.text);
}
```
