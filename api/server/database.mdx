## Overview

The `Database` struct provides persistent storage for accounts, characters, quests, skills, inventory, bank vaults, and all player progress. It uses SQLite with WAL mode for concurrent access and automatic migrations.

## Struct Definition

```rust
pub struct Database {
    pool: SqlitePool,
}
```

Wraps a connection pool for async database operations.

## Constructor

```rust
pub async fn new(database_url: &str) -> Result<Self, sqlx::Error>
```

Creates a new database connection pool and runs migrations.

<ParamField path="database_url" type="&str" required>
  SQLite database URL (e.g., `"sqlite://game.db"` or `"sqlite::memory:"`)
</ParamField>

<ResponseField name="database" type="Result<Database, sqlx::Error>">
  Database instance with connection pool
</ResponseField>

**Configuration:**
- Journal mode: WAL (Write-Ahead Logging) for better concurrency
- Max connections: 5
- Auto-runs migrations on startup

**Example:**
```rust
let db = Database::new("sqlite://aeven.db").await?;
```

<ParamField path="username" type="&str" required>

</ParamField>

<ParamField path="password" type="&str" required>
  Plain text password (will be hashed)
</ParamField>

<ResponseField name="account_id" type="Result<i64, String>">

</ResponseField>

**Security:**
- Passwords hashed with Argon2id
- Salted with random 128-bit salt
- UNIQUE constraint on username (case-sensitive)

- HP, prayer points, mana

- Interior entrance position
- Bank contents and gold
- Played time (incremented by delta)

**Performance:**
- Called periodically (e.g., every 30 seconds) and on logout
- Single UPDATE query with all fields

## Database Schema

### accounts
```sql
CREATE TABLE accounts (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    username TEXT UNIQUE NOT NULL,
    password_hash TEXT NOT NULL,
    created_at TEXT DEFAULT CURRENT_TIMESTAMP,
    last_login TEXT
)
```

### characters
```sql
CREATE TABLE characters (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    account_id INTEGER NOT NULL,
    name TEXT UNIQUE NOT NULL,
    gender TEXT NOT NULL DEFAULT 'male',
    skin TEXT NOT NULL DEFAULT 'tan',
    hair_style INTEGER,
    hair_color INTEGER,
    x REAL DEFAULT -30.0,
    y REAL DEFAULT 19.0,
    hp INTEGER DEFAULT 10,
    prayer_points INTEGER,
    mp INTEGER,
    gold INTEGER DEFAULT 0,
    -- Equipment slots
    equipped_head TEXT,
    equipped_body TEXT,
    equipped_weapon TEXT,
    -- ... more equipment
    inventory_json TEXT DEFAULT '[]',
    skills_json TEXT,
    bank_json TEXT DEFAULT '[]',
    bank_gold INTEGER DEFAULT 0,
    bank_max_slots INTEGER DEFAULT 50,
    current_map TEXT,
    played_time INTEGER DEFAULT 0,
    is_admin BOOLEAN DEFAULT FALSE,
    FOREIGN KEY (account_id) REFERENCES accounts(id)
)
```

### character_quests
```sql
CREATE TABLE character_quests (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    character_id INTEGER NOT NULL,
    quest_id TEXT NOT NULL,
    state TEXT NOT NULL DEFAULT 'active',
    objectives_json TEXT DEFAULT '{}',
    started_at TEXT DEFAULT CURRENT_TIMESTAMP,
    completed_at TEXT,
    FOREIGN KEY(character_id) REFERENCES characters(id),
    UNIQUE(character_id, quest_id)
)
```

### arena_stats
```sql
CREATE TABLE arena_stats (
    character_id INTEGER PRIMARY KEY,
    total_wins INTEGER DEFAULT 0,
    total_matches INTEGER DEFAULT 0,
    total_kills INTEGER DEFAULT 0,
    total_deaths INTEGER DEFAULT 0,
    current_streak INTEGER DEFAULT 0,
    best_streak INTEGER DEFAULT 0,
    total_gold_won INTEGER DEFAULT 0,
    FOREIGN KEY(character_id) REFERENCES characters(id)
)
```

## Example Usage

```rust
use std::sync::Arc;

// Initialize database
let db = Arc::new(Database::new("sqlite://aeven.db").await?);

// Create account
let account_id = db.create_account("player1", "secure_password").await?;

// Verify login
if let Some(account) = db.verify_account_password("player1", "secure_password").await {
    println!("Login successful! Account ID: {}", account.id);
}

// Create character
let character = db.create_character(
    account_id,
    "Steve",
    "male",
    "tan",
    Some(2),
    Some(4),
).await?;

// Save character progress
db.save_character(
    character.id,
    character.x,
    character.y,
    character.hp,
    character.prayer_points,
    character.mp,
    &character.skills,
    character.gold,
    &character.inventory_json,
    character.equipped_head.as_deref(),
    // ... more fields
).await?;

// Load quest state
let quest_state = db.load_character_quest_state(character.id).await?;
println!("Active quests: {}", quest_state.active_quests.len());

// Update arena stats
db.update_arena_stats(character.id, true, 3, false, 100).await?;
```
